# 🐧 Linux Daily Master (Linux命令每日精通)

**Linux Daily Master** 是一款基于 Edge/Chrome 浏览器的 **New Tab（新标签页）** 扩展插件。

它的核心理念是利用碎片化时间，在每次打开新标签页时展示一个 Linux 命令知识卡片。同时，它兼顾了工作效率，采用了“每日仅自动弹出一次”**的策略(该策略暂时没在该版本体现)，并提供了一个美观实用的**沉浸式桌面模式，确保不会打扰您的日常浏览体验。

## ✨ 核心功能

### 1. 智能学习卡片
*   **精美 UI**：采用磨砂玻璃（Glassmorphism）设计风格，悬浮于动态壁纸之上。
*   **详细内容**：包含命令名称、简述、详细参数解释、代码示例（支持一键复制）。
*   **智能频率**：**每天首次**打开浏览器时自动弹出学习卡片，后续打开新标签页默认收起卡片，不干扰工作。

### 2. 沉浸式桌面 (Desktop Mode)
*   **无干扰体验**：当学习卡片关闭时，展示极简的桌面。
*   **实用组件**：包含当前时间、日期显示。
*   **聚合搜索**：内置搜索框，支持 Bing、Google、Baidu 快速切换，直接开始网络搜索。
*   **快捷唤起**：右下角提供悬浮球，随时点击（或按空格键）重新唤起学习卡片。

### 3. 知识库系统 (Library)
*   **全览模式**：全屏查看所有命令列表。
*   **状态管理**：实时查看“已收藏”、“已学会”和“未学习”的命令。
*   **搜索与筛选**：支持在知识库中快速搜索特定命令。
*   **进度重置**：支持一键重置所有学习进度。

### 4. 交互与反馈
*   **快捷键支持**：`Space` (空格键) 快速切换 卡片/桌面 模式。
*   **微交互**：收藏动画、复制反馈。
*   **记忆系统**：基于 LocalStorage 本地存储学习进度，保护隐私。

## 📂 工程目录结构

```text
linux-command-daily/
│
├── manifest.json          # 核心配置文件 (Manifest V3)
├── background.js         # 后台服务脚本 (用于生命周期管理)
├── README.md             # 项目说明文档
│
├── assets/               # 静态资源目录
│   ├── commands.json     # 核心数据：Linux命令数据库
│   └── icons/            # 插件图标文件
│       ├── icon16.png
│       ├── icon48.png
│       └── icon128.png
│
└── popup/                # 前端界面代码
    ├── index.html        # 新标签页主结构 (包含桌面层、卡片层、知识库层)
    ├── style.css         # 样式表 (CSS3, 动画, 磨砂效果)
    └── script.js         # 业务逻辑 (交互, 存储, 每日判断, 渲染)
```

## 🛠️ 数据结构说明

命令数据存储在 `assets/commands.json` 中，格式如下：

```json
[
  {
    "id": 1,
    "command": "ls -la",
    "description": "列出目录下所有文件（包括隐藏文件）的详细信息",
    "explanation": "<strong>-l</strong>: 长格式显示<br><strong>-a</strong>: 显示所有文件",
    "category": "文件操作",
    "difficulty": "初级",
    "example": "ls -la ~/Documents"
  }
  // ... 更多命令
]
```

## 🚀 部署与安装指南

由于本插件尚未发布到应用商店，需要通过**开发者模式**手动安装。

### 1. 准备工作
1.  下载本项目源码。
2.  确保 `assets/icons/` 目录下有三个图标文件 (`icon16.png`, `icon48.png`, `icon128.png`)。如果缺失，请自行找任意 PNG 图片重命名放入即可，否则无法加载。

### 2. 在 Edge 浏览器中安装
1.  在地址栏输入：`edge://extensions/` 并回车。
2.  开启左侧菜单（或页面上方）的 **“开发人员模式”** 开关。
3.  点击 **“加载解压缩的扩展”** 按钮。
4.  选择本项目的根目录文件夹 (`linux-command-daily`)。
5.  安装成功后，打开一个新的标签页（Ctrl+T）即可体验。

### 3. 在 Chrome 浏览器中安装
1.  在地址栏输入：`chrome://extensions/` 并回车。
2.  开启右上角的 **“开发者模式”** 开关。
3.  点击左上角的 **“加载已解压的扩展程序”**。
4.  选择本项目的根目录文件夹。

## 📖 使用说明

1.  **初次见面**：每天第一次打开浏览器或新标签页时，卡片会自动弹出。
2.  **学习操作**：
    *   点击 **“我学会了”**：标记为已掌握，明天不再主动推送此题。
    *   点击 **“明日再来”**：放入复习队列，下次优先展示。
    *   点击 **“❤️”**：加入收藏夹。
3.  **进入工作模式**：
    *   点击卡片右上角的 **“X”**。
    *   点击底部的 **“返回桌面”** 按钮。
    *   或者直接按下键盘 **空格键 (Space)**。
    *   *此时将显示时钟和搜索框，您可以像往常一样使用浏览器。*
4.  **查看进度**：点击卡片左上角的 **图书图标** 打开知识库，管理您的所有卡片。

## ⚙️ 技术栈

*   **HTML5 / CSS3**: 使用 Flexbox 布局，CSS Variables 管理主题，Keyframes 实现平滑动画。
*   **JavaScript (ES6+)**: 原生 JS 实现，无第三方框架依赖（轻量级）。
*   **Chrome Extension API**:
    *   `chrome.storage.local`: 用于持久化存储用户进度、设置和上次弹出日期。
    *   `Manifest V3`: 符合最新的浏览器扩展安全标准。
